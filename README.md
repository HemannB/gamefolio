# Gamefólio — Ghostwire: Tokyo

Um site portfólio fictício desenvolvido para apresentar o jogo **Ghostwire: Tokyo**, com foco em prática de HTML, CSS e JavaScript.

Acesse o projeto online:  
https://gamefolio-web.vercel.app

---

##  Sobre o projeto

O objetivo deste projeto é criar uma página web organizada, visualmente atraente e funcional, simulando a apresentação de um jogo.

O site foi inspirado na estética do próprio jogo, trazendo:

- atmosfera sombria com elementos neon  
- layout limpo e organizado  
- navegação simples e intuitiva  
- foco em experiência do usuário  

---

## Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript  

---

## Funcionalidades

- Navegação com scroll suave entre seções  
- Botão "voltar ao topo"  
- Carrossel de imagens interativo  
- Seção de trailer com vídeo incorporado  
- Layout responsivo com unidades relativas  
- Efeitos visuais com hover
